### 今日の学び

**技術的な発見:**
1. **スプレッド演算子の落とし穴**
   ```typescript
   // ❌ 大きな配列でStack Overflow
   String.fromCharCode(...array)
   
   // ✅ 安全な方法
   for (let i = 0; i < array.length; i++) {
     binary += String.fromCharCode(array[i]);
   }
   ```
